Git Crash course

git touch : Easiest way to create empty file

git init : Initialized empty git repository

git status : To check the status of current branch/repo

git add _file_name_ : Add files to repo

git add *.html : Add any html files to staging area

git rm --cahced _file_name_ : Remove files from staging area

git commit : This will open up VM editor, then you can add comment and save

To skip the above, You can use git commit -m '//comment'

git igone : will use to mention the file that we don't want to include to our repo
